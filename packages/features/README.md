# @peas/features

Feature-specific components that build on top of `@peas/ui`.

## Usage

```tsx
import { SomeFeature } from "@peas/features";
```

## Structure

- `src/components/` - Feature components organized by domain
- `src/styles.css` - Feature-specific styles

## Development

```bash
# Build the package
yarn build

# Watch for changes
yarn watch

# Type check
yarn typecheck
```
